--------------------------------------------------------------------------------
FONCTIONNALITÉS DE BASE
--------------------------------------------------------------------------------

[✓] Question 1 : Étude du code HTML
    Le code HTML a été étudié et compris. La structure comprend :
    - Un terrain de jeu (#terrain)
    - Un bouton "Une cible" (#btn-single-target)
    - Une zone de contrôle avec champ numérique et bouton "Démarrer"
    - Un affichage du nombre de cibles restantes (#remaining-count)
    - Un chronomètre (#chrono avec #chrono-minutes, #chrono-seconds, #chrono-tenths)

[✓] Question 2 : Gestion du bouton "Une cible"
    RÉALISÉ COMPLÈTEMENT
    - Création d'un élément cible avec la classe 'target'
    - Positionnement aléatoire dans le terrain (fonction positionTarget)
    - Ajout comme enfant de #terrain (fonction addTargetToField)
    - Gestion du clic avec effet visuel (classe 'hit')
    - Disparition après 1 seconde (setTimeout)
    
    Fonctions implémentées :
    - createTarget() : crée un élément div.target
    - positionTarget(target) : positionne aléatoirement la cible
    - addTargetToField(target) : ajoute la cible au terrain
    - handleTargetClick(event) : gère le clic sur la cible

[✓] Question 3 : Gestion du chronomètre
    RÉALISÉ COMPLÈTEMENT
    - Démarrage du chronomètre avec setInterval (100ms)
    - Incrémentation de la variable globale 'time'
    - Affichage au format MM:SS:D (minutes:secondes:dixièmes)
    - Arrêt et réinitialisation du chronomètre
    
    Fonctions implémentées :
    - startChrono() : démarre le timer
    - stopChrono() : arrête le timer
    - resetChrono() : remet à zéro
    - updateChronoDisplay() : met à jour l'affichage

[✓] Question 4 : Déroulement du jeu
    RÉALISÉ COMPLÈTEMENT
    - Bouton "Démarrer" fonctionnel
    - Suppression des cibles existantes au début
    - Réinitialisation du chronomètre
    - Création du nombre de cibles demandé
    - Mise à jour du compteur de cibles restantes
    - Détection de la fin du jeu (toutes les cibles détruites)
    - Arrêt du chronomètre à la fin
    
    Fonctions implémentées :
    - startGame() : démarre une nouvelle partie
    - endGame() : termine le jeu
    - updateRemainingCount() : met à jour le compteur
    - clearAllTargets() : supprime toutes les cibles

--------------------------------------------------------------------------------
INSTRUCTIONS D'UTILISATION
--------------------------------------------------------------------------------

1. Ouvrir le fichier index.html dans un navigateur web moderne
   (Chrome, Firefox, Safari, Edge)

2. Pour créer une seule cible :
   - Cliquer sur le bouton "Une cible"
   - Cliquer sur la cible pour la détruire

3. Pour jouer une partie complète :
   - Choisir le niveau de difficulté (Facile, Moyen, Difficile)
   - Entrer le nombre de cibles souhaité (1-50)
   - Cliquer sur "Démarrer"
   - Cliquer sur toutes les cibles le plus rapidement possible
   - Entrer votre nom pour enregistrer votre score

4. Le Hall of Fame affiche automatiquement les 10 meilleurs scores
